## Система автоматизации заказов для ресторанов

### Tasks:
* #### Интерфейс заказов:
	* Бронирование столиков
	* Возможность просмотра меню и выбора необходимого продукта в корзину
	* Возможность выбрать самовывоз и необходимое кафе из существующих, вместо доставки
	* Просмотр статуса своих заказов
	* Возможность войти в свой аккаунт
	* Возможность заказа без звонка оператору, но можно сделать уточнение звонком
* ### Интерфейс для ресторана
	* Возможность вносить заказы вручную
	* Просмотр заказов
	* Возможность выставлять статус заказа
	* Возможность изменить меню с помощью админ панели
	* Если мы делаем доставку, то необходим расчет ее стоимости ( Можно отправлять запрос на доставку с оценкой цены)
	* Сохраняем все заказы в базу данных и ведем учет финансов
